============================================================
  怪物猎人荒野 启动崩溃修复工具
  Monster Hunter Wilds Crash Fix
============================================================

【怎么用】

  1. 把【整个 MHW-CrashFix 文件夹】放进怪物猎人荒野游戏根目录
     (就是 MonsterHunterWilds.exe 所在的那个文件夹)

     最终位置应该长这样:
        ...\MonsterHunterWilds\
            ├── MonsterHunterWilds.exe
            └── MHW-CrashFix\        ← 本文件夹
                    ├── fix.bat       ← 修复(双击)
                    ├── restore.bat   ← 回滚(双击)
                    └── ...其他文件

  2. 双击本文件夹里的  fix.bat

  3. 等黑色窗口显示 "[OK] 修复操作已全部执行",然后去启动游戏。


【修复了什么】

  解决游戏启动就崩溃的问题 (EXCEPTION_ILLEGAL_INSTRUCTION)。
  根本原因:本地的着色器缓存 (shader.cache2) 内容损坏,
  或 Mesh Shader 编译路径在特定 CPU 上生成了不支持的指令。
  本工具会:备份配置 → 清掉损坏的缓存 → 关闭 Mesh Shader。


【重要提醒】

  · 首次启动游戏会明显变慢 (1~3 分钟),是在重新编译着色器,
    属正常现象,不是卡死。

  · 所有操作都可回滚:双击 restore.bat 即可恢复。

  · 修复完后这个文件夹可以删掉,不删留着也不碍事。


【放错位置了?】

  如果双击 fix.bat 提示"找不到 MonsterHunterWilds.exe",
  说明文件夹没放对。请确认它放在和 .exe 同一个文件夹里。
